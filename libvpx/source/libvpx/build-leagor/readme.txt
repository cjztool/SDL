======iOS======
1. On Mac OS X, run "terminal", enter <libvpx>/build-leagor.
2. Run build-ios.sh. It will generate libvpx.a at build-leagor.

Remark
  It include armv7, armv7s, arm64.

======Windows======
1. run cygwin. enter <libvpx>/build-leagor.
2. Run build-win-ia32.sh. It will generate vpx.sln at build-win-x86.
